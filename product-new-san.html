Key Features

Pocket-Sized

The SAN-60 is barely larger than a credit card — take spectrum analysis anywhere.

FPGA Engine

Dedicated FPGA-based digital signal processing for real-time capture without missed signals.

Rich APIs

C/C++, C#, Python, Java, MATLAB, LabVIEW, GNU Radio, and SCPI support.

Expansion

Optional IO expansion boards, trigger I/O, and GNSS modules.

Cross-Platform

Run on Windows or Linux, ARM or x86 — from Raspberry Pi to desktop workstations.

USB Powered

Single USB-C cable provides data and power — truly plug-and-play operation.
