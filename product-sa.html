Key Features

Embedded Ready

Ultra-compact form factors designed for integration into automated test systems.

Signal Generator

SAM-60 includes optional integrated analog signal generator (100 kHz–6.3 GHz).

Low Power

SAM-60 uses just 7–10 W — ideal for battery-powered field deployments.

Extended Temp

Optional extended temperature range (−40°C to 70°C) for harsh environments.

OCXO Option

Optional oven-controlled crystal oscillator for ≤0.15 ppm frequency drift.

API Integration

Full API for C/C++, Python, MATLAB, LabVIEW, and GNU Radio.
