Key Features

Ethernet Native

Gigabit Ethernet for true remote operation — monitor spectrum from anywhere on the network.

Built-in GNSS

Integrated multimode GNSS with 1PPS output for time-stamped and geolocated measurements.

Signal Gen Option

NXM-60 includes an optional analog signal generator (100 kHz – 6.3 GHz).

Wide Temp Range

Operates from −20°C to 70°C with optional extended temperature ranges.

4G Data Module

Optional 4G data module for remote deployments without Ethernet infrastructure.

GNSS-Disciplined OCXO

Optional GNSS-disciplined OCXO for ppb-level frequency stability.
